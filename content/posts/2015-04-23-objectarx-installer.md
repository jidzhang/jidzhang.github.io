> ObjectARX 是 AutoCAD 二次开发的 SDK。以下为历史版本下载链接，部分链接可能已失效，可到 [Autodesk Developer Center](https://aps.autodesk.com/developer) 查找最新版本。

## 各版本下载

| AutoCAD 版本 | 下载链接 |
|-------------|---------|
| 2010 (32/64) | [ObjectARX 2010](http://download.autodesk.com/akdlm/esd/dlm/objectarx/ObjectARX_2010_Win_64_and_32Bit.exe) |
| 2009 (32/64) | [ObjectARX 2009](http://download.autodesk.com/esd/objectarx/2009/ObjectARX_2009_Win_64_and_32Bit.exe) |
| 2008 32-bit | [ObjectARX 2008 32](http://download.autodesk.com/esd/objectarx/2008/ObjectARX_2008_32Bit.exe) |
| 2008 64-bit | [ObjectARX 2008 64](http://download.autodesk.com/esd/objectarx/2008/ObjectARX_2008_64Bit.exe) |
| 2007 Core | [Arx_Core 2007](http://download.autodesk.com/esd/objectarx/2007/Arx_Core.exe) |
| 2007 SDK | [Arx_All 2007](http://download.autodesk.com/esd/objectarx/2007/Arx_All.exe) |
| 2006 Core | [Arx_Core 2006](http://download.autodesk.com/WebPub/autocad/oarx2006/Arx_Core.exe) |
| 2006 SDK | [Arx_All 2006](http://download.autodesk.com/WebPub/autocad/oarx2006/Arx_All.exe) |
| 2004 Core | [Arx_Core 2004](http://download.autodesk.com/WebPub/autocad/oarx/arx_core.exe) |
| 2000 Core | [ObjectArxCore 2000](http://download.autodesk.com/pub/objectarx/ObjectArxCore.exe) |

> 2007 及更早版本分 Core（基础库）和 SDK（完整开发包）两个下载。2008 起合并为单一安装包。

## 提示

- 新版本建议从 [Autodesk ADN](https://aps.autodesk.com/developer) 获取
- ObjectARX 需要 Visual Studio 对应版本（如 2010 对应 VS2010）
- 安装后需配置 include/lib 路径到 VS 项目中
